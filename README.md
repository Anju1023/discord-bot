# 🎉 AnjuBot

An adorable Discord bot made with 💖 by Anju.  
楽しく、そしてちょっぴり便利な機能を提供するDiscord用ボットです！

---

## 🚀 Features / 機能一覧

- 🧠 AI風の返答
- 🧡 絵文字リアクション
- 🔁 自動メッセージ対応　などなど！

---

## 📦 Installation / インストール方法

1. Node.js をインストール（v18以降推奨）
2. このリポジトリをクローン：
   ```
   git clone https://github.com/AnjuDev/AnjuBot.git
   ```
3. 依存パッケージをインストール：
   ```
   npm install
   ```
4. `config.json` を設定！

---

## 🧑‍💻 Usage / 使い方

```bash
node index.js
```
